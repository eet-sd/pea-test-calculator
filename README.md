# ค่าทดสอบเพื่อการตรวจรับพัสดุ กฟภ. (PEA Acceptance Test Calculator)

เว็บแอปแบบ static สำหรับตอบคำถาม: พัสดุรหัสนี้/ชื่อนี้ ซื้อจำนวนเท่านี้ → ต้องทดสอบหัวข้ออะไรบ้าง สุ่มกี่ตัวอย่าง ค่าทดสอบประมาณเท่าไร

## ข้อจำกัด

- ราคา/จำนวนสุ่มเป็นไปตาม snapshot ของชีต — ตรวจสอบกับหน่วยงานทดสอบก่อนใช้อ้างอิงจริง
- รหัสที่ชีตระบุจำนวนสุ่ม "ไม่กำหนด" หน้าเว็บจะให้กรอกจำนวนสุ่มเอง
- กรณีสุ่มเกิน 3 ตัวอย่างของกลุ่ม VT/CT (RMTR-021/2553) คำนวณให้แล้วใน `rules.js` (แยก 3 ตัวอย่างทดสอบ withstand/PD/inter-turn ที่เหลือทดสอบ accuracy & markings) — ใช้ได้เฉพาะกรณีจำนวนสุ่มตรงตามตาราง (5/8/13/20/32) ถ้ากรอกจำนวนสุ่มเองเป็นค่าอื่นจะคิดแบบทดสอบครบทุกตัวอย่าง
